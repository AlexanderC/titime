# titime - [T]ool for [I]nstant [T]ime [T]racking

`titime` is the very basic and simple to use tool for tracking your time

## Prerequisites

- Linux/MacOS

## Installation and Usage

Installation:

```bash
npm install
```

Development:

```bash
npm run start
```

Create distribution:

```bash
npm run make
```

Publish to GitHub:

```bash
npm run publish
```
